# Lesson Content Rules

- **No grammar terminology** — strip "singular", "plural", "feminine", "Form X", "prefix", "verbal noun", "occurs N times". Use plain English about meaning.
- **Hook text = language, not meaning** — the note under each phrase explains the word/root/pattern being taught, NOT what the verse means.
- **Root letter format** — always show both Arabic and English: `**أ ل ه** (alif lām hā)` — mobile students can't read Arabic well.
- **Headings: single word form** — `### 3 · كُبْرَى (greatest)` not multi-word phrases.
- **Anchor headings** — `### ⭐ · Anchor Phrase` — no Arabic in the heading itself.
- **Surah references at end** — `(Al-Baqarah 2:34)` after the audio tag, never before Arabic text.
- **Practice phrases** — ordered shortest → longest (progressive difficulty).
- **Learn phrases** — as short as possible while retaining the story hook.
- **Story Context Principle** — include enough text so the phrase has a story the student can hook into (who said what to whom). But trim anything beyond the story hook.
- **Translation style** — simple everyday English, not academic. See `docs/LESSON-PLAN.md` → "Translation Style".
