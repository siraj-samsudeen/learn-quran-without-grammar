Issue UI — Three Options

Each card below shows the same verse (3:18) with a different issue flagging UI. Click chips to toggle (interactive).

Option A — Inline Chip Row

Issue chips sit in a dedicated row at the bottom of every card, always visible. Click a chip to flag, type a note inline. No extra clicks to open/close.

Option B — Corner Flag Button + Dropdown

Small flag icon in the top-right corner. Click to open a dropdown panel. When flagged, a summary badge appears below the remark. Closest to the old picker's approach.

Option C — Subtle Issue Bar (Recommended)

A quiet grey bar at the bottom of each card with a ⚐ icon and faded chips. Unobtrusive when empty, lights up warmly when an issue is flagged. Single click to flag — no open/close ceremony. Note field appears inline.
