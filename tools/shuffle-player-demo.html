🔀 Quranic Sentence Shuffle Player

Learn Quran Without Grammar — Audio Review Mode

How to test this demo

1. Build lesson audio first:
python tools/build-lesson-audio.py tools/lesson-audio/test-3-sentences.yaml -o /tmp/test-lesson-audio

2. Serve this file with a local server from the project root:
cd /path/to/project && python -m http.server 8000

3. Open http://localhost:8000/tools/shuffle-player-demo.html

The player loads manifest.json from the lesson audio directory and plays sentences in random order. Each sentence is an Arabic clip followed by a 2-second pause and the English translation.

Multi-lesson support

Add more manifests to the manifests array to combine sentences from multiple lessons into one shuffle pool.
