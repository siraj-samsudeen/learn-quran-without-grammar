How It Works

Per-lesson workflow

  1. SCORE LLM scores verses (story, familiarity, teaching_fit). Already done for all 7 lessons.
  2. PICK Open the Picker HTML, review algorithm's auto-picks, adjust selections, click "Save JSON". The JSON saves to the lesson folder as selections.json.
  3. SIGNAL DONE Tell Claude: "I'm done with L3 picks" — or edit teacher/pipeline-status.json and set "picking": "done".
  4. WRITE Claude generates the lesson markdown from your selections.
  5. TAMIL Tamil translations added and reviewed.
  6. AUDIO Claude builds YAML, fetches timestamps, generates TTS, builds MP3s. Progress visible in terminal.
  7. REVIEW Structural + pedagogical review. Teacher signs off.
  8. PUBLISH Commit + push → live in ~1 minute.
